# Tic-Tac-Toe Game

I developed this Tic-Tac-Toe game in C++ to strengthen my understanding of programming fundamentals and game logic. The game allows two players to play on a 3×3 board and automatically detects a winner or a draw.

## Features
- Two-player gameplay
- 3×3 game board
- Winner detection
- Draw detection
- Prevents players from choosing an occupied position

## Skills I Used
- C++
- Functions
- 2D Arrays
- Loops
- Conditional Statements
- Basic Game Logic
